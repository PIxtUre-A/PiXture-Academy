เรียนฟิสิกส์สบายตา
โฟกัสเนื้อหาแบบมินิมอล

รวม เนื้อหา + คลิป + โจทย์ + เฉลย ในหน้าเดียว เปลี่ยนแท็บได้ไว ไม่หลุดโฟกัส

เหตุผลที่เหมาะกับการเรียน

  • โทนสีสบายตา อ่านนาน ๆ ได้
  • โครงสร้างชัดเจน: หน้าแรก → เลือกบท → เนื้อหา/คลิป/โจทย์/เฉลย
  • ทำงานได้ในไฟล์เดียว เหมาะกับ GitHub Pages
  • บันทึก “บทที่กำลังเรียนล่าสุด” ไว้ในเครื่องคุณ (ไม่ต้องล็อกอิน)
